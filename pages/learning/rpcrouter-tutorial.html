Learn the RPC router

To use the RPC router, you first need to download the RPC router on the downloads page. Then, import it in your project and reference it through a <script> tag.
Now, create a script tag as usual. Inside it, create a new WorldRPCRouter following the WorldRouter model. Then, use the addActionRoute method to create an Action Route. It is just a route (e.g. /sayHello), but instead of loading HTML it loads a javascript function. 

<script>
    // Initialize the router
    const router = new WorldRPCRouter();

    // Define a route and associate it with a function
    router.addActionRoute('/sayhello', sayHello);

    // Example function to be executed when /sayhello is accessed
    function sayHello() {
        alert('Hello!');
    }

    // Example of navigating to the route programmatically
    document.addEventListener('DOMContentLoaded', () => {
        document.getElementById('helloButton').addEventListener('click', () => {
            router.navigateTo('/sayhello');
        });
    });
</script>
